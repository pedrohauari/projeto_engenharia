# 🧮 Portal de Ferramentas Matemáticas com Streamlit

* 🖼️ Acesse o portal abaixo:
* [![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://projetoengenharia.streamlit.app/)


## 🛠️ Ferramentas Disponíveis

| Ferramenta | Tipo | Tecnologia Base | Descrição |
| :--- | :--- | :--- | :--- |
| **Achador de Raízes** | Numérico | SciPy (Newton-Raphson) | Encontra raízes reais com alta precisão e visualização gráfica. |
| **Integral Simbólica** | Simbólico | SymPy | Resolve integrais indefinidas e fornece a solução exata. |
| **Gráfico de Funções** | Visual | Matplotlib/Plotly | Plotagem interativa de funções complexas. |
| **Derivada Numérica** | Numérico | NumPy | Cálculo de taxa de variação via diferenças finitas. |
| **Derivada Simbólica** | Simbólico | Sympy | Derivada Simbólica com o Sympy.|
| **Integral Numérica** | Numérico | Scipy | Cálculo Numérico de Integrais definidas. |
| **Calculadora de Expressões Matemáticas** | Numérico | NumPy | Expressões Simbólicas resolvidas Numericamente.|


## 🖥️ Tecnologias
* **Python 3**
* **SymPy** - Computação Simbólica
* **Scipy** - Integração Numérica
* **Numpy** - Computação Numérica
* **Matplotlib** - Construção de Gráficos
* **Streamlit** - Interface web

## 🚀 Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone [https://github.com/pedrohauari/portal-matematico.git](https://github.com/pedrohauari/portal-matematico.git)

2. Instale as Dependencias e Rode o Portal:
```bash
    pip install -r requirements.txt
    streamlit run Home.py
```
